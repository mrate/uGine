# uGine

**!! WIP !!**

Educational Vulkan 3D engine with editor.

Features
 - Vulkan abstraction layer
 - Lua scripting
 - Forward+ shading
 - Shadows
 - Postprocess
 - SSAO
 - Skeletal animation (in compute shader)
 - Raycasting
 - ...
